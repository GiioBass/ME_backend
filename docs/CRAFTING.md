# Sistema de Crafteo y Recetas

Este documento detalla los objetos que el jugador puede fabricar, los materiales necesarios y las herramientas requeridas.

## 1. Estaciones de Trabajo
- [ ] **Inventario (Básico):** Crafteo simple sin necesidad de mesa (ej. Antorchas, Vendas).
- [ ] **Mesa de Trabajo:** Permite crear herramientas, armas de madera/piedra y muebles básicos.
- [ ] **Horno de Fundición:** Procesa minerales en lingotes y cocina alimentos.
- [ ] **Yunque:** Necesario para armas y armaduras de metal avanzado.
- [ ] **Mesa de Alquimia:** Para crear pociones y transmutar materiales.

## 2. Recetas Básicas (Inventario)
- [ ] **Antorcha:** 1 palo + 1 carbón.
- [ ] **Vendas:** 2 telas.
- [ ] **Cuerda:** 3 fibras vegetales.

## 3. Herramientas y Armas (Mesa de Trabajo / Yunque)
### Herramientas
- [ ] **Pico de Piedra:** 3 piedras + 2 palos (Mesa de Trabajo).
- [ ] **Hacha de Hierro:** 3 lingotes de hierro + 2 palos (Yunque).

### Armas
- [ ] **Espada de Hierro:** 2 lingotes de hierro + 1 palo (Yunque).
- [ ] **Arco de Caza:** 3 palos + 3 hilos (Mesa de Trabajo).
- [ ] **Bastón Mágico:** 2 maderas nobles + 1 gema (Mesa de Trabajo).

## 4. Alquimia y Consumibles
- [ ] **Poción de Curación:** 1 botella de agua + 2 hierbas rojas (Mesa de Alquimia).
- [ ] **Poción de Maná:** 1 botella de agua + 2 hongos azules (Mesa de Alquimia).
