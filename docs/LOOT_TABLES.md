# Tablas de Botín (Loot Tables)

Probabilidades de obtener objetos de enemigos y cofres.

## 1. Enemigos
### Slime Verde
- [ ] **Gelatina de Slime:** 60% (Material de crafteo).
- [ ] **Poción Menor incompleta:** 10%.
- [ ] **Moneda de Cobre:** 30% (1-3 monedas).

### Esqueleto Guerrero
- [ ] **Hueso:** 50%.
- [ ] **Espada Oxidada:** 5% (Arma, baja durabilidad).
- [ ] **Escudo Roto:** 5%.

## 2. Cofres
### Cofre de Madera (Bosque/Superficie)
- [ ] **Monedas:** 100% (10-50 oro).
- [ ] **Pociones:** 50% (1-2 Pociones de Vida).
- [ ] **Mapa del Área:** 20%.

### Cofre Dorado (Jefe de Mazmorra)
- [ ] **Objeto Mágico Garantizado:** 100% (Arma o Armadura Rara).
- [ ] **Gran cantidad de oro:** 100% (200-500 oro).
- [ ] **Gema Preciosa:** 30%.
