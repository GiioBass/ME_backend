# Sistema de Habilidades y Clases

Este documento define las habilidades activas y pasivas de cada clase, así como su progresión.

## 1. Luchador (Fighter)
**Estilo:** Tanque / Daño físico cuerpo a cuerpo.
- **Habilidades Activas:**
    - [ ] **Golpe Pesado (Nivel 1):** Daño extra + probabilidad de aturdir. Coste: 10 Energía.
    - [ ] **Grito de Guerra (Nivel 3):** Aumenta el ataque propio y de aliados cercanos por 3 turnos.
    - [ ] **Bloqueo Perfecto (Nivel 5):** Reduce el daño del próximo ataque recibido a 0.
- **Pasivas:**
    - [ ] **Constitución de Hierro:** +10% HP máximo.

## 2. Tirador (Marksman)
**Estilo:** Daño a distancia / Críticos / Trampas.
- **Habilidades Activas:**
    - [ ] **Disparo Preciso (Nivel 1):** Daño alto, siempre acierta. Coste: 15 Energía.
    - [ ] **Lluvia de Flechas (Nivel 3):** Daño de área ligero.
    - [ ] **Trampa de Osos (Nivel 5):** Inmoviliza al enemigo si la pisa.
- **Pasivas:**
    - [ ] **Ojo de Águila:** +5% probabilidad de crítico.

## 3. Mago (Mage)
**Estilo:** Daño mágico explosivo / Control / Utilidad.
- **Habilidades Activas:**
    - [ ] **Bola de Fuego (Nivel 1):** Daño alto de fuego. Coste: 20 Maná.
    - [ ] **Escudo de Maná (Nivel 3):** Absorbe daño a costa de maná.
    - [ ] **Rayo de Hielo (Nivel 5):** Daño medio + ralentiza al objetivo.
- **Pasivas:**
    - [ ] **Sabiduría Arcana:** Regeneración de maná aumentada.
